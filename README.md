# Pumpkin DEX

# 🚀 Quick Start

📄 Clone or fork `github repo`:

```sh
git clone https://github.com/Bild96/pumpkin-dex.git
```

💿 Install all dependencies:

```sh
cd pumpkin-dex
yarn install
yarn start
```

🌐 live demo
``` 
https://pumpkin-dex.netlify.app
http://pumpkin-dex-a38004.spheron.app

```
